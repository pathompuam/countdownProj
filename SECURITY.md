# Security Policy

## Supported Versions

Currently, the following versions of Lux Countdown are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please open an issue in the GitHub repository. While we don't have a private reporting process yet, we take all reports seriously and will address them as quickly as possible.
